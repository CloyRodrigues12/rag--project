# RAG-Powered Website Chatbot

## Description
An AI assistant that ingests any URL, recursively scrapes content, and uses Retrieval-Augmented Generation (RAG) to answer user questions based on the collected data.

## Solution Approach
This full-stack application utilizes a React frontend for the user interface and a Django backend for managing API requests. Supabase (PostgreSQL + pgvector) is used to handle both structured relational data and unstructured vector embeddings to ensure minimal latency during RAG retrieval.

## Dependencies
* Python 3.x
* Django & Django REST Framework
* Supabase-py
* Node.js & React
